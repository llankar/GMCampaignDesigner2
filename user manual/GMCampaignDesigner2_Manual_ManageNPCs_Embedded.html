GMCampaignDesigner2 User Manual

🧙 Feature: Manage NPCs (Non-Player Characters)

The "Manage NPCs" section allows you to build a cast of non-player characters with portraits, backstories, secrets, affiliations, and item ownership. This is essential for worldbuilding and consistent storytelling.

By using right mouse button over a NPC, a menu appears, it allows to delete the NPC or display the npc portrait on the second screen

🧭 Where to Find It

Click the fourth icon from the top on the left-side panel to open the NPC Manager.

📋 NPC List View

The interface is nearly identical to the PC manager and includes the following columns:

  • Name, Role, Background
  • Secret, Traits, Factions, Objects

Use the Search box to find NPCs by keyword. Add new NPCs using the Add button or edit existing ones by double-clicking their entry.

💾 Saving & Loading NPCs

At the bottom right of the NPC list window, use Save NPCs to save a json of all the npcs or Load NPCs to load a json list of NPCs.

➕ Creating or Editing an NPC

At the top of the editor:

  • Portrait: Set a visual for the character by uploading an image or using AI generation (if available).
  • Name / Role: Fill out the basic identity details.
  • Background: Use rich text formatting to flesh out the NPC’s history or public persona.

Scroll down to enter:

  • Secret: Hidden knowledge or secret motivations, editable via a rich text field.
  • Traits: Personality features, habits, or noteworthy details.

At the bottom:

  • Factions: Click Add Faction to associate the NPC with one or more groups.
  • Objects: Assign inventory using Add Object.
  • Click Save to apply changes or Cancel to discard.

📝 Notes

  • NPCs and PCs share similar structure but are tracked separately.
  • Use Secrets and Factions to create interwoven intrigue and power dynamics.
  • Portrait images can dramatically improve player immersion and memory.
