The "Manage PCs" panel allows you to create, view, and edit Player Characters (PCs) used in your campaign. Each character can have a portrait, background, secrets, traits, faction affiliations, and owned items.

âž• Adding or Editing a PC

Click Add (top-right) or double-click a row to open the PC Editor.

The top of the editor includes a portrait area and input fields:

  • Select Portrait: Choose an image from your files.
  • Create Portrait with Description: Generate an AI-based image (if supported).
  • Name / Role: Basic identity of the character.

Background / Secret / Traits fields include a rich text editor:

  • Style options: Bold, Italic, Underline, etc.
  • Lists and alignment tools are also available.

Finally, you can assign the PC to:

  • Factions: Click "Add Faction" to associate the PC with one or more factions.
  • Objects: Assign items the PC owns via "Add Object".

Click Save to confirm changes or Cancel to discard.

ðŸ’¾ Saving & Loading

Use the Save PCs and Load PCs buttons in the bottom right of the main list view to persist or refresh the PC data.

ðŸ“ Notes

  • All PC data is saved to the currently active campaign database.
  • Portraits are stored as file paths or base64 depending on configuration.
  • Use consistent naming and faction tagging to organize large rosters efficiently.
